# calorie-counter
기존에 존재하는 'fatSecret'이라는 앱을 즐겨썼던 사용자입장으로서,  개선되었으면 좋겠는점들을 추가해보는 서비스

## 개요

calorie-counter 서비스는 기본적으로 사용자가 섭취한 음식을 입력하면 음식의 영양정보를 제공하는 서비스이며, 사용자들은 건강한 식단관리를 할 수있다는 장점이 있습니다.


## 사용 기술
* JDK 17
* Spring Boot 2.7.7
* MyBatis 3
* MySQL 8.0
